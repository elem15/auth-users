# User authentication service.

## Description

Based on [Nest](https://github.com/nestjs/nest) framework and PostgreSQL Data Base.  
You can register, edit, delete user profile. JWT access and refresh tokens are used for authentication.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
